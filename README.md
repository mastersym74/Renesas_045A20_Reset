# Renesas 045A20 Battery Reset

Цей проект містить Arduino Nano скетч для роботи з контролером батареї Renesas 045A20 через SMBus.  
Програма зчитує статус батареї та, у разі встановлення PF-флагу (Permanent Failure), надсилає команду для скидання.

## Файли

- `renesas_045A20_reset.ino` — основний Arduino код.
- `SMBus_Schematic.txt` — схема підключення Arduino Nano до батареї.

## Підключення

- Підключити SDA батареї до A4 (Arduino Nano) з підтягуючим резистором 4.7-10 кОм до 3.3В або 5В.
- Підключити SCL батареї до A5 (Arduino Nano) з підтягуючим резистором 4.7-10 кОм до 3.3В або 5В.
- Підключити землю батареї до GND Arduino.

?? **Не підключайте живлення батареї до Arduino!** Використовуйте тільки лінії даних.

## Використання

1. Завантажити скетч у Arduino Nano.
2. Відкрити Serial Monitor зі швидкістю 9600 бод.
3. Програма просканує пристрої, зчитає статус батареї і спробує скинути PF-флаг, якщо він встановлений.
